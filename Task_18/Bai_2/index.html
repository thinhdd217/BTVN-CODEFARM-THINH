Câu hỏi 1: HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu dùng để tạo cấu trúc cho trang web. Nó định nghĩa các thành phần như tiêu đề, đoạn văn, liên kết, hình ảnh, bảng biểu, v.v.

Câu hỏi 2: CSS có tác dụng gì?

CSS (Cascading Style Sheets) dùng để định dạng và trang trí giao diện cho trang web. Nó kiểm soát màu sắc, font chữ, khoảng cách, bố cục, hiệu ứng hover, responsive design, v.v.

Câu hỏi 3: JavaScript làm được gì?

JavaScript là ngôn ngữ lập trình chạy phía client (trình duyệt) giúp trang web trở nên tương tác: xử lý sự kiện click, validate form, gọi API, thay đổi nội dung động, tạo animation, và nhiều tính năng phức tạp hơn khi kết hợp với các framework như React, Vue,...

Câu hỏi 4: Accordion hoạt động như thế nào?

Accordion sử dụng JavaScript để toggle class "active" khi click vào tiêu đề. Khi có class active, phần trả lời sẽ có max-height lớn hơn và padding, kết hợp với CSS transition tạo hiệu ứng mở/đóng mượt mà.
